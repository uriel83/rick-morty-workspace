Rick and Morty Character Collection
אפליקציית Angular לניהול אוסף דמויות מ־Rick and Morty: טעינה מה־API, יצירה/עריכה מקומית, מועדפים (LocalStorage), אינסוף גלילה, וספריית UI ייעודית.

תכולה עיקרית
Angular + Standalone Components + Signals

RxJS: switchMap, forkJoin, catchError

Reactive Forms עם preview לתמונה ולוקיישן

LocalStorage למועדפים (Persistence אוטומטי)

Infinite Scroll

ספריית UI: PopupComponent, CharacterCardComponent

רכיבים ושירותים
HomeComponent: רשימת דמויות, אינסוף גלילה, פופאפים למחיקה/טופס.
FavoritesComponent: מציג דמויות מועדפות (טעינה לפי IDs, forkJoin + catchError) והסרה.


CharacterFormComponent: טופס ריאקטיבי עם preview, שדות סטטוס/מין/מיקום (טעינת כל המיקומים מה־API).


CharactersService: קריאות ל־/character ו־/location עם טיפוסים גנריים ו־switchMap לטעינת כל דפי המיקומים.


FavoritesService: Signal של רשימת IDs, התמדה ל־LocalStorage, עדכונים איממוטביליים עם set/update.

API חיצוני
Rick & Morty API – https://rickandmortyapi.com/api


דמויות: /character?page=N, /character/{id}


מיקומים: /location?page=N

RxJS – דגשים
טעינת כל המיקומים: switchMap לעמוד ראשון → חישוב מספר עמודים → forkJoin על כל העמודים → map ל־flat.


טעינת מועדפים: forkJoin לקריאה מקבילה לפי IDs, עם catchError(() => of(null)) וסינון בתוצאת המנוי.


Change Detection – OnPush
האפליקציה משתמשת ב־ChangeDetectionStrategy.OnPush בכל קומפוננטות התצוגה לשיפור ביצועים והפחתת רינדורים מיותרים.
 ניהול ה־state נעשה באמצעות Angular Signals עם עדכונים איממוטביליים (set/update), מה שמבטיח שהתצוגה תתרענן אוטומטית גם תחת OnPush.
